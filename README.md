## 📌 Overview

LIFELINE AI is an AI-powered productivity companion that helps users plan, prioritize, and complete tasks before deadlines are missed. Unlike traditional to-do list applications, Lifeline AI actively assists users throughout the entire workflow by converting unstructured thoughts into actionable plans, intelligently scheduling work, predicting deadline risks, and providing personalized AI coaching.

The application is designed for students, professionals, entrepreneurs, and anyone struggling with procrastination, overwhelming workloads, and time management.

---

## 🎯 Problem Statement

**The Last-Minute Life Saver**

Existing productivity tools primarily function as passive reminder systems that notify users about deadlines but fail to actively help them complete their work. Users often experience procrastination, cognitive overload, poor prioritization, and missed deadlines.

LIFELINE AI addresses this challenge by transforming productivity from passive reminders into intelligent execution and accountability.

---

## ✨ Key Features

* 📝 **Messy Dump AI** – Convert unstructured thoughts into organized tasks.
* ⚠️ **Predictive Risk Assessment** – Identify tasks most likely to miss deadlines.
* 🧩 **Intelligent Task Decomposition** – Break large tasks into manageable subtasks.
* 📅 **Dynamic Timeline Scheduler** – Automatically organize daily schedules.
* 🤖 **Four AI Productivity Coaches**

  * 🔥 Ignis – High-energy accountability coach.
  * 🌿 Zephyr – Mindfulness and stress reduction.
  * 📊 Vera – Logical planning and prioritization.
  * 💙 Amis – Emotional support and encouragement.
* 📈 Personalized productivity recommendations.
* 📱 Modern responsive user interface.

## 🚀 Getting Started

### Clone the repository

```bash

```


### Install dependencies

```bash
npm install
```

```bash
npm run dev
```
